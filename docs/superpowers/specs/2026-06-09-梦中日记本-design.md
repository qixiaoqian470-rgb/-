# 「梦中日记本」— 完整设计规格文档

> 设计日期：2026-06-09 | 版本：v1.0
> 灵感来源：秒秒Guo × Penderecki's Garden

---

## 一、产品概述

「梦中日记本」是一款 AI 语音交互日记 PWA 应用。用户上传照片 → 照片转化为墨色粒子点彩画 → 与 AI 温柔对话 → 自动生成诗意日记 → 滑动浏览记忆碎片画廊。

**核心情感**：白底墨韵，东方禅意，记忆如墨滴在水中慢慢晕开。

## 二、技术架构

### 2.1 技术栈

| 层 | 技术 |
|---|------|
| 前端框架 | 纯 HTML + CSS + JS（ES Modules），无构建工具 |
| AI 接口 | Google Gemini API（gemini-2.5-flash） |
| 语音转文字 | 浏览器 Web Speech API（免费、离线可用） |
| 语音朗读 | 浏览器 SpeechSynthesis API |
| 存储 | IndexedDB（日记数据）+ localStorage（设置） |
| 粒子渲染 | Canvas 2D + Perlin Noise |
| 背景音乐 | Web Audio API + 内置音频文件 |
| 部署 | 静态文件，本地服务器（`npx serve`）+ PWA |

### 4.2 字体系统

| 层级 | 字体 | 大小 | 字重 |
|------|------|------|------|
| 品牌名 | Noto Serif SC（衬线体） | 15px | 400 |

### 4.3 布局

- 桌面端：最大宽度 420px 居中，模拟手机体验
- 移动端：全宽，底部安全区域适配
- 信息密度低，大量留白
- 不做网格、不做多层卡片

### 4.4 动效

| 场景 | 动画 | 时长 |
|------|------|------|
| 粒子有机摇摆 | Perlin Noise 持续漂浮 | 持续 |
| 录音中麦克风 | 黑色光晕脉冲（box-shadow） | 2s 循环 |
| 页面切换 | 淡入淡出（opacity） | 300ms |
| 卡片 hover | 上浮 4px + 阴影加深 | 200ms |
| AI 生成中 | 打字机逐字出现 | ~3-5s |
| 气泡进入 | opacity + translateY(12→0) | 300ms |
| 加载骨架屏 | 呼吸闪烁 | 1.5s 循环 |

## 六、粒子系统设计

### 6.1 核心理念

> 粒子 = 图像本身。照片上传后，通过采样像素颜色，用 Canvas 2D 点彩画法重建图像。粒子有微弱的 Perlin Noise 有机浮动。

### 6.2 技术方案

| 项目 | 选择 |
|------|------|
| 渲染引擎 | Canvas 2D |
| 粒子数量 | 1000-3000（根据照片复杂度自适应） |
| 粒子形状 | 圆形，径向渐变边缘（硬中心 → 柔消退） |
| 粒子大小 | 2-6px，亮部小暗部大 |
| 颜色采样 | 从上传照片像素网格采样 |
| 运动 | 简化 Perlin/Simplex Noise，振幅 ±2-5% |

### 6.3 粒子行为

```
每个粒子：
├── 锚点位置：像素采样确定的原始坐标
├── 当前偏移：Perlin Noise 驱动的实时位移
├── 颜色：从照片像素采样（在我们的白底版本中
│         转为灰度 → 不同浓度的黑色粒子）
├── 大小：亮部=小粒子(2-3px)，暗部=大粒子(4-6px)
├── 透明度：0.3-0.9（暗部浓、亮部淡）
└── 边缘：径向渐变（rgba 中心不透 → 边缘全透）

运动层：
├── 层1：低频大振幅（0.1-0.3Hz）→ 模拟整体飘动
├── 层2：高频微振幅（1-3Hz）→ 模拟微颤
└── 音频联动：背景音乐音量 → 振幅调制
```

### 6.4 图像 → 粒子转化流程

```
1. 用户上传照片
2. 将照片绘制到离屏 Canvas
3. 网格采样（如 50×60 = 3000 个采样点）
4. 每个采样点读取像素 RGB
5. 转为灰度值 gray = 0.299R + 0.587G + 0.114B
6. 映射：
   - 位置：采样网格坐标 + 微小随机抖动
   - 颜色：rgba(0,0,0, 1-gray)  // 亮部透明、暗部浓黑
   - 大小：2 + (1-gray) * 4     // 亮部小、暗部大
7. 存储粒子数组，Canvas 渲染
8. 每帧 Perlin Noise 更新偏移 → generateDiary
```

### 6.5 粒子切换日记

- 首页粒子区支持触摸/鼠标左右滑动
- 每次滑动 → 粒子渐隐 → 加载下一篇日记的粒子快照 → 渐显
- 过渡动画 0.5s opacity

## 七、AI 对话设计

### 7.1 Gemini System Prompt

```
你是一位温暖、细腻的文学编辑。用户会给你一段口语化的语音转写文本，
你需要将其转化为一篇精美的日记。

请按以下 JSON 格式返回（只返回 JSON，不要其他内容）：

{
  "title": "诗意的标题（10字以内）",
  "content": "润色后的日记正文，温暖文学的语调，保留原意的同时让文字更优美",
  "tags": ["情绪标签1", "情绪标签2", "情绪标签3"],
  "mood": "整体情绪（开心/平静/忧伤/焦虑/期待/感动/思念/迷茫）",
  "quote": "从日记中提取一句最有感触的话"
}

要求：
1. 保持第一人称，不要改变用户原意
2. 风格温暖治愈，像村上春树或吉本芭娜娜的散文语调
3. 适当分段，让阅读节奏舒适
4. 标签要准确反映情绪，不要泛化
```

### 7.2 AI 聊天对话

- 在日记详情页正文下方，自然过渡到 AI 对话
- 用户可以就这篇日记继续和 Claude 对话
- 对话上下文包含日记内容
- 聊天 Prompt：以温柔朋友的身份，对日记内容进行回应、提问或安慰

## 八、音频设计

### 8.1 语音输入

- 使用浏览器 `SpeechRecognition` API（Web Speech API）
- 支持中文（`lang: 'zh-CN'`）
- 实时转文字显示
- 最长录制 5 分钟
- 失败降级：提示用户手动输入文字

### 8.2 语音朗读（TTS）

- 使用浏览器 `SpeechSynthesis` API
- 日记详情页提供"听日记"按钮
- 支持中文声音选择

### 8.3 背景音乐

- 预设轻音乐（内置）：
  - 🌧️ 雨声白噪音
  - 🎹 轻柔钢琴
  - 🎸 木吉他
  - 🔥 壁炉噼啪声
- Web Audio API 播放，循环
- 音量通过 ☰ 菜单调节
- 页面切换不中断

### 8.4 音频 → 粒子联动

- `AnalyserNode` 分析背景音乐
- RMS 音量 → 粒子摇摆幅度
- 低频段 → 大粒子位移
- 高频段 → 小粒子颤动

## 九、错误处理

| 场景 | 处理 |
|------|------|
| 麦克风权限被拒 | "无法访问麦克风，请在浏览器设置中允许麦克风权限" + 手动输入降级 |
| Web Speech 不支持 | 检测 `SpeechRecognition` API，不支持则提示手动输入 |
| Gemini API Key 未设置 | 引导用户进入设置面板填入 Key |
| Gemini API 调用失败 | "AI 暂时无法回应，请稍后重试" + 重试按钮 |
| IndexedDB 不可用 | 降级到 localStorage（提示容量有限） |
| 网络中断 | "网络连接中断，请检查网络" + 离线缓存（SW） |
| 照片上传失败 | "图片加载失败，请尝试其他照片" |
| 粒子渲染性能 | 低端设备自动降低粒子数（检测 FPS） |

## 十、边缘情况

- 空日记列表 → 首页粒子区显示默认画面 + "点击新建你的第一篇日记"
- 超长日记文本 → 卡片内滚动
- 极暗/极亮照片 → 粒子灰度映射拉伸到全范围
- 小尺寸照片 → 自适应采样网格
- 快速连续点击 → 防抖 500ms
- localStorage 满 → IndexedDB 自动替代
- Gemini API 配额耗尽 → 提示等待 + 本地缓存未完成日记
