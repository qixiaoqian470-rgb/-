## 需求讨论

### 用户意图
用户想做一个类似的语音交互应用。

### 提出的问题
1. 想做成什么形态？（网页 / 桌面应用 / 微信小程序）
2. 语音识别方案选择

### 关键冲突
- 用户选了"微信小程序"，但微信小程序**不支持浏览器 Web Speech API**
- 微信小程序运行在微信私有 WebView 中，无法使用标准浏览器语音 API

### 建议的替代方案：PWA 网页应用

| 特性 | 微信小程序 | PWA 网页应用 |
|------|-----------|-------------|
| 像小程序一样的聊天界面 | ✅ | ✅ |
| 浏览器语音 API | ❌ 不支持 | ✅ |
| 桌面/手机都能用 | 仅微信内 | ✅ 浏览器打开 |
| 添加到桌面 | ❌ | ✅ 可安装为桌面应用 |
| 后端 AI (Gemini) | ✅ | ✅ |
| 开发复杂度 | 需注册审核 | 零门槛 |
